`@media`: Diseño Responsivo

Este texto cambiará el tamaño de su fuente y el layout de los elementos cuando la pantalla sea más pequeña. Es la base del diseño que se adapta a móviles, tabletas y escritorios.

`@font-face`: Fuentes Personalizadas

Este párrafo utiliza una fuente personalizada cargada con @font-face.

La fuente se llama "Playfair Display" y debería verse elegante.

`@supports`: Capacidades del Navegador

Si tu navegador soporta display: flex, este texto tendrá un color de fondo diferente para demostrarlo.

Esto permite aplicar estilos progresivos, mejorando la experiencia en navegadores modernos sin romper los antiguos.
