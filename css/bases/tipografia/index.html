Fuentes Personalizadas y Alternativas

Este párrafo usa la fuente **Roboto**, una fuente sans-serif moderna y legible. Si Roboto no carga, el navegador usará una fuente sans-serif genérica como Arial o Helvetica.

Este párrafo utiliza la fuente **Playfair Display**, una fuente serif con un toque elegante. Es ideal para titulares o textos que buscan un estilo más clásico.

Aquí vemos un ejemplo de fuente **monospace** genérica. Cada carácter ocupa el mismo ancho, perfecto para bloques de código o datos. console.log("¡Hola Mundo!");

Truncamiento de Texto con Elipsis

Este es un texto muy largo que se truncará con puntos suspensivos si no cabe en el ancho asignado. Es útil para descripciones cortas o titulares.

Función clamp()

Esta técnica permite que el tamaño del texto se adapte al tamaño de la pantalla, sin necesidad de media queries. Usar clamp() es ideal para mejorar la legibilidad en móviles y monitores grandes.
