Diseños Web: Fluid, Responsive y Adaptive

Explorando cómo las páginas se adaptan a diferentes dispositivos.

1. Diseño Fluido (`fluid`)

Este contenido utiliza anchos en porcentajes. Redimensiona la ventana del navegador para ver cómo el texto y las imágenes se ajustan proporcionalmente. No hay "saltos" abruptos, simplemente una escala continua.

Box 1 (50% ancho)
Box 2 (40% ancho)

Observa cómo los elementos se escalan de forma continua. En pantallas muy pequeñas o muy grandes, esto puede no ser óptimo.

2. Diseño Responsive (`responsive`)

Este diseño combina la fluidez con "breakpoints" (puntos de quiebre) definidos por media queries. Redimensiona la ventana: en ciertos anchos, el layout cambiará significativamente. Esto es el estándar moderno.

Item 1
Item 2
Item 3
Item 4

Arrastra el borde de la ventana. Verás que las columnas cambian de 4 a 2 y luego a 1 en puntos específicos, mientras se mantienen fluidas entre esos puntos.

3. Diseño Adaptativo (`adaptive`)

El diseño adaptativo salta entre layouts fijos en puntos de quiebre específicos. No hay fluidez entre ellos; el layout es más "rígido" para cada breakpoint. Este ejemplo simula un layout para escritorio, tablet y móvil.
