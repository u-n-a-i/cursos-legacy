Animación con Motion Path en CSS

Moviendo elementos a lo largo de rutas SVG.

1. Animación Simple a lo Largo de una Línea

Un círculo se mueve de izquierda a derecha a lo largo de una línea recta.

2. Animación a lo Largo de una Curva Compleja

Un cuadrado sigue una ruta curva personalizada con un efecto de rebote.

3. `offset-rotate` y `offset-anchor`

El elemento rota automáticamente para seguir la ruta y su punto de anclaje se ajusta.

4. Múltiples Elementos en la Misma Ruta

Varios círculos siguen la misma ruta, pero con diferentes retrasos.

5. Uso de un `path` desde SVG Externo/Interno

Una flecha sigue una ruta compleja definida en un SVG.
